# 爪爪鍵盤 Clawboard

爪爪鍵盤 Clawboard — 為可支援嘸蝦米使用者打造的智慧輸入法，結合傳統字根輸入的高效率與現代 AI 技術的智慧化，讓您的中文打字體驗更快速、更聰明。

無論是日常聊天、商務郵件或專業寫作，Clawboard 都能成為您最得力的輸入助手。

## 字根檔案產生

以嘸蝦米字根為例，字根表格檔案（`clawroot_table.bin`）可透過以下網址產生：

**[https://dimanai.github.io/clawboard-usage/](https://dimanai.github.io/clawboard-usage/)**

## 核心特色

### ■ 智慧記憶詞頻預測

- 基於 Bigram 模型的下一字預測
- 自動學習您的輸入習慣
- 越用越聰明，大幅提升輸入速度
- 支援刪除修正機制，避免錯誤學習
- 隱私優先：所有資料僅儲存在您的裝置上

### ■ AI 智慧寫作助手

- 校正：即時偵測並修正錯別字
- 優化：讓句子更自然、通順、好讀
- 專業：一鍵轉換為正式、具說服力的內容
- 翻釋：智慧中英互譯
- 簡短：精簡冗詞，保留核心意思
- 支援 OpenAI、Claude、Gemini 多種 AI 服務

### ■ 支援嘸蝦米輸入法核心

- 即時候選字顯示
- 智慧按鍵提示（視覺回饋可用字根）
- 流暢的組字體驗
- 支援簡體輸入
- 支援打繁出簡模式
- 支援速打模式

### ■ 快速詞庫管理

- 自訂常用詞彙快速輸入
- 支援匯入/匯出詞庫
- 在主應用中輕鬆管理

### ■ 便捷編輯工具

- 空白鍵長按啟動游標快速橫移
- 一鍵剪下、複製、貼上
- 全形/半形標點快速切換
- 內建計算機模式，隨時運算

### ■ 精緻使用體驗

- 標準 QWERTY 排列，符合打字習慣
- 按鍵預覽效果
- Shift / Caps Lock 狀態清楚標示
- 流暢的動畫與觸覺回饋

## 隱私保護

- 所有輸入資料與學習記錄僅儲存在您的裝置上
- AI 功能需自行設定 API Key，您完全掌控資料流向
- 不收集、不上傳任何個人資訊
- 開源透明，值得信賴

## 完美整合 iOS 生態系統

- 支援所有支援第三方鍵盤的 App
- 與 iOS 原生功能無縫整合
- 輕量化設計，不占用過多儲存空間

## 適合誰使用？

- 嘸蝦米輸入法使用者
- 需要快速且準確中文輸入的專業人士
- 重視隱私的使用者
- 希望結合 AI 提升寫作品質的創作者
- 追求效率的商務人士

## 使用提示

1. 首次使用請在「設定」→「一般」→「鍵盤」→「鍵盤」中新增「爪爪鍵盤」
2. 在主 App 的「設定」頁面開啟「記憶詞頻預測」功能
3. 若需使用 AI 功能，請在設定中填入您的 API Key
4. 空白鍵長按可啟動游標快速移動模式
